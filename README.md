# AdHoc Engine

### On Windows ###
```
  git clone --recursive https://github.com/Filippo-BSW/AdHoc.git
  cd AdHoc && mkdir build && cd build
  ..\install.bat Release
```

### On macOS ###
```
  git clone --recursive https://github.com/Filippo-BSW/AdHoc.git
  cd AdHoc && chmod +x install.sh && mkdir build && cd build
  ../install.sh Release
```
