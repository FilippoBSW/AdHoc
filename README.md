# AdHoc Engine

### On Windows ###
```
  git clone --recursive https://github.com/Filippo-BSW/AdHoc.git
  cd AdHoc && mkdir build && cd build
  ..\Install\install_windows.bat Release
```

### On macOS - Linux ###
```
  git clone --recursive https://github.com/Filippo-BSW/AdHoc.git
  cd AdHoc && sh build.sh (Mac or Linux) (Debug or Release)
```
